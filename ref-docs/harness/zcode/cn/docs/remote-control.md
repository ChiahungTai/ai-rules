Remote Control
Remote Control 用来把当前桌面端 ZCode 工作区临时开放给手机访问。你可以在手机上查看当前任务、输入指令、继续推进 Agent 会话；真正的代码修改、命令执行和项目环境仍然保留在桌面端。
适合什么场景
任务已经在桌面端跑起来，需要离开电脑后继续看进度。
临时收到问题反馈，需要用手机快速追加一条指令。
长任务执行中，需要随时检查 Agent 当前状态。
手机扫码连接
在 ZCode 左下角侧栏点击手机图标，打开 移动端远程控制。
弹窗左侧会生成当前工作区的二维码和连接地址。用手机扫码，或复制链接在手机浏览器中打开，即可进入这个工作区。
连接成功后，手机端会接入当前桌面工作区。桌面端仍然是实际运行环境，手机端负责查看会话、输入消息和触发操作。
手机端能做什么
手机端适合处理轻量但关键的操作：
查看当前 Agent 回复与任务进度。
输入新的需求或补充上下文。
在长任务中继续追问、确认或中止。
从外部场景快速回到当前工作区。
注意：同一时间只支持一个手机页面连接，且手机端只能访问桌面端当前已打开的工作区。
手机端界面
连接后首先看到的是工作区面板。顶部显示当前的连接状态；下面列出这台设备上已打开的工作区和任务，右侧标签是每个任务的状态。点击任意任务即可进入会话，也可以用项目栏右侧的加号直接新建任务，底部的转发按钮则用来把当前界面截图分享出去。
进入任务后就是会话界面。顶部是当前任务和它所属的项目；Agent 的回复都在这里，展开「已工作」一栏可以查看每一步的执行过程，回复下方的图标用来复制、反馈或分享。要继续推进任务，直接在底部输入框补充需求，Agent 会接着执行。
