使用统计
使用统计用来查看 ZCode 的实际使用情况。它分为 应用用量 和 编程套餐 两个部分：前者统计本地 ZCode 会话，后者读取 Z.ai / BigModel Coding Plan 的远端额度与消耗。
应用用量
应用用量 统计当前设备上的 ZCode 会话记录，适合回看自己最近在 ZCode 里完成了多少任务。
你可以看到：
Token 用量、会话数、消息数和活跃天数。
当前连续使用天数、最长连续使用天数和峰值时段。
按天 Token 趋势、活跃热力图和模型用量排行。
按天 Token 趋势图支持悬停查看每日各模型的用量明细；下方的模型用量环图则汇总各模型的 Token 占比，方便确认主力模型的实际消耗分布。
右上角可以切换 全部时间、最近 30 天、最近 7 天，用于快速对比近期使用情况。
编程套餐
编程套餐 读取 Z.ai / BigModel Coding Plan 的远端统计，适合查看套餐额度、模型消耗和工具调用情况。下图以 Z.ai - Coding Plan 为例；如果你使用的是 BigModel - Coding Plan，同样可以在这里查看对应的额度和消耗。
这里会展示：
5 小时 prompt 池、每周额度、MCP 每月额度等套餐状态。
GLM-5.2、GLM-5-Turbo 等模型的 Token 消耗。
联网搜索 MCP、网页读取 MCP 等工具调用次数。
点击或展开额度信息时，还可以查看当前上下文窗口的组成、消息和工具调用占比，以及 5 小时、每周、工具调用等剩余额度。
