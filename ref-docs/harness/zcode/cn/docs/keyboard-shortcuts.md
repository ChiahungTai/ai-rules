快捷键表
这里保留的是 ZCode 里最常用、最值得记住的快捷键。我们优先整理和任务流、聊天输入、日常高频操作直接相关的内容，不把系统原生菜单里的低频项全部堆上来。
全局操作
Windows 按键Mac 按键描述
Ctrl+NCommand+N新建任务
Ctrl+BCommand+B切换侧栏
Ctrl+JCommand+J切换终端面板
Ctrl+Alt+BOption+Command+B切换右侧面板
Ctrl+KCommand+K打开命令中心
Ctrl+FCommand+F查找对话
Ctrl+[Command+[后退到上一个任务或工作区
Ctrl+]Command+]前进到下一个任务或工作区
Ctrl+Shift+[Shift+Command+[切换到上一个对话
Ctrl+Shift+]Shift+Command+]切换到下一个对话
