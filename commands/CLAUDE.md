# Claude Code Commands

## 🎯 用途

這個目錄包含自定義的 Claude Code slash commands。Claude Code 會自動讀取此目錄中的所有 `.md` 檔案作為可用的 slash commands。

## 📋 使用方式

### 基本語法
```bash
/command-name [參數] [內容]
```

### 主要命令
- `/explain` - 解釋技術概念
- `/lessons` - 提取知識精華

## 📁 目錄結構

直接在此目錄中新增 `.md` 檔案即可創建新的 slash command。

**重要**: 不需要創建符號連結，Claude Code 會自動識別此目錄中的檔案。
